# C - Variadic functions

The aim of this project is to learn about variadic functions in **C language**.

## Technologies
* Scripts written in Bash 5.0.17(1)
* All files compiled on Ubuntu 20.04 LTS using gcc
* Vi Editor

## Files
Scripts written in C:

| Filename | Description |
| -------- | ----------- |
| `0-sum_them_all.c` | Returns the sum of all its parameters |
| `1-print_numbers.c` | Prints numbers |
| `2-print_strings.c` | Prints strings |
| `3-print_all.c` | Prints anything |
| `variadic_functions.h` | Header file with all the prototypes of all functions used |
